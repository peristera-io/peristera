root `adr/`; the conventions Kamara follows (personal-data metadata,
OpenFGA, audit, search, permalinks, transactional storage) are root ADRs
0007, 0009–0012, 0015. Decisions here are Kamara-specific.

Same rules as the root: copy the process, keep it short, immutable once
accepted (a change of mind is a new ADR).
