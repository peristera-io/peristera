Peristera European workplace platform
Early — building in public

A sovereign home for your organisation's work.

Peristera is an open-source, federated workplace suite — files, tasks, and documents — built to give European organisations a real alternative to Microsoft 365. Self-host it on your own infrastructure, or let a provider run it for you. Your data stays yours.

View the code Read the strategy

What's in the suite

Kamara — files

A per-tenant file store with content-addressed, at-rest-encrypted storage and a clean browser experience.

Ergonomos — tasks

Task and work tracking that talks to the rest of the suite, so a file attaches to a task without leaving your tenant.

Documents in the browser

Open and edit office documents directly in Kamara, each tenant on its own isolated engine — no shared document server.

Federated by design

Organisations connect to each other directly, peer to peer — collaboration without a central gatekeeper.

Why Peristera

  • Sovereignty. Runs on European infrastructure, or your own. No dependency on a US hyperscaler to do your daily work.
  • No lock-in. Open source (AGPL). Your data and the software that runs it are both yours to keep.
  • Isolation you can trust. Every tenant is its own walled space — separate identity, separate data, enforced by the platform, not by policy.
  • Built to be run by many. Self-hosters first, then managed-service providers, then a hosted service — the same code everywhere.

Where it is today

Peristera is early and honest about it. The platform runs — identity, per-tenant isolation, files, tasks, and browser document editing are real and working — and it is being built in the open, nights and weekends. If that resonates, the code is the best place to start.
